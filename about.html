About



This website is a place where we are compiling all the TNT cannon‍-‍related technology we find, in order to have an easily searchable database for anybody to access. This project is open source and is being developed by a group of volunteer contributors.


You can check out the Github Page  for the (messy) source code, the raw data, and (eventually) info about contributing to this project.

For discussion, updates, suggestions, questions or whatever else, be sure to visit the Discord server!
